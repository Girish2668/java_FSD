ordered list

it is used to display a list in a specific ordered
ol tag is used to create an ordered list

the default order is number(1,2,3...)

type attribute helps to change the default order(type="A", "a", "i", "I", "1")

reversed attribute helps to reverse the order

start attribute helps to start the list from the specific number
