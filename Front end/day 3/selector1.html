selector

it is used to select HTML elements for styling
  • element selector
  • id selector
  • attribute selector
  • grouping selector
  • combined selector
    • descendant selector
    • child selector
    • adjecent selector
    • general sibling selector
    • universal selector

elements selector :

it is used to select elements by their tag name
        syntax:   tagName{
                        property : value;
        }

Id selector :

it is used to select elements ny their id

id is an attribute used to opening tag

id should be unique

id selector indicated with # 

            syntax:   #idName{
                           property: value;            }

Class selector :

it is used to select elements by their Class

class is an attribute used in opening tag

class name should be duplicate

class selector indicated with .(Dot) 
            syntax:   .className{
                           property: value;
            }

Grouping selector :

it is used to select elements as grouping

grouping elector indicated with ,(cama)

it is used to apply same style to multiple elements

        syntax:    selector1,selector2,selector3{
                      property: value;
        }

combined selector

it is used to combine multiple selector

it is used to apply same style to multiple elements

descendent selector

it is used to select that are descendantaf the a specidiedelement

used to select both direct and indirect child element

descendant selector indicated with a space 

        synatx: parent child{
                    property: value;
        }

child selector

it is used to select elements that are diret children of a specified element

used to select only direct child elements

child selector indicated with >(greater than symbol)

        syntax: parent>child;
               property:value;

Adjacent Slibling selector

it is used to select elements tjat are adjecent sibling of a specified element

used to select only the next sibling elements

adjacent sibling selector indicated with + (plus) 

        syntax: element1+element2{
                      property:value;
        }

general Slibling selector

it is used to select elements tjat are general sibling of a specified element

used to select all sibling elements

adjacent sibling selector indicated with ~ (tagol) 

        syntax: element1~element2{
                      property:value;
        }

Universal Selector

It is used to select all elements

Universal selector indicated with *

        syntax:       *{
                        property: value;
      Note : it is recommended to have 
        padding :0;
        margin:0;
        box-sizing: border-box;
        content-box-it is used to set the box model by default
                   -it calculates the width and height of an element
                    including padding and border
                    ex: width: 100px;
                        height: 100px;
                        padding: 10px;
                        border: 2px solid black;
                        box-sizing: content-box;
                    calculation:
                        width: 100px + 10px + 2px +2px = 112px;
                        height: 100px + 10px + 2px+2px = 112px;


                 box-sizing: border-box-it is used to set the box model by manually
                -it calculates the width and height of an element 
                including padding and border and provides the exact width and height
                ex: width: 100px;
                    height: 100px;
                    padding: 10px;
                    border: 2px solid black;
                    box-sizing: border-box;
                calculation:
                    width: 100px + 10px + 2px = 100px;
                    height: 100px + 10px + 2px = 100px;
